freezing.css

freezing.css is a CSS microframework, inspired by Joqd project, first Iranian front-end framework. freezing.css is not as large and complete as Joqd, but it still can help in creating fast websites! I actually wrote this micro-framework for my own projects, but I decided to make it public, because people out there may want to use a responsive micro-framework, and of course freezing.css is one of the best choices for a fast, simple and beautiful responsive website!


If you want to start working with freezing.css, just go and clone our repository, github.com/prp-e/freezing.css. you also can use the link to the freezing.css file, and add copy it to your HTML document for best performance.
freezing.css is licensed under MIT license. It means you can copy, fork and use this code, contribute to it and also make your own version. Feel free to have your own customized freezing.css!
If you can't use freezing.css, or you find any bugs in this framework, feel free to open an issue on github and let met know what that bug is. You will help me improve my product, and of course, your products will be improved!
